---
title: AWS のサービス・特徴を学習する
description: AWS の各種サービスとそれぞれの特徴を押さえておく
---

AWS にどういったサービスがあり、それぞれのサービスにはどういった特徴があるのかを以下一覧で掴んでおく。

サービスのイメージが掴めない場合は、サービス名をキーワード検索して、複数ページをざっくり読み、イメージや要点を掴む。

![AWS のサービス一覧](./aws-services.jpg)

## コンピューティング

| サービス                                                        | 概要                                                       |
| :-------------------------------------------------------------- | :--------------------------------------------------------- |
| [Amazon EC2](http://aws.amazon.com/jp/ec2/)                     | クラウド内で提供される仮想サーバ                           |
| [AWS Lambda](http://aws.amazon.com/jp/lambda/)                  | サーバーのプロビジョニングや管理をすることなくコードを実行 |
| [Elastic Beanstalk](http://aws.amazon.com/jp/elasticbeanstalk/) | ウェブアプリの実行と管理                                   |

## コンテナ

| サービス                                                    | 概要                                   |
| :---------------------------------------------------------- | :------------------------------------- |
| [Elastic Container Registry](http://aws.amazon.com/jp/ecr/) | コンテナイメージの保存、管理、デプロイ |
| [Elastic Container Service](http://aws.amazon.com/jp/ecs/)  | スケーラブルなコンテナ実行方法         |

## ストレージ

| サービス                                                    | 概要                                 |
| :---------------------------------------------------------- | :----------------------------------- |
| [S3](http://aws.amazon.com/jp/s3/)                          | クラウド内のスケーラブルなストレージ |

| [S3 Glacier](http://aws.amazon.com/jp/glacier/)             | クラウド内のアーカイブストレージ     |

| [EBS](http://aws.amazon.com/jp/ebs/)                        | EC2 のブロックストレージサービス     |

| サービス                                                                 | 概要                                                                         |
| :----------------------------------------------------------------------- | :--------------------------------------------------------------------------- |
| [VPC](http://aws.amazon.com/jp/vpc/)                                     | 独立したクラウドリソース                                                     |

| [Global Accelerator](http://aws.amazon.com/jp/global-accelerator/)       | 世界中のユーザーに提供するアプリケーションの可用性とパフォーマンスを改善する |
| [Elastic Load Balancing](http://aws.amazon.com/jp/elasticloadbalancing/) | 複数のターゲットにわたる着信トラフィックの分配                               |
