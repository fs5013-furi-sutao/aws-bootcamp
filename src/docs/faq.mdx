## AWS 認定クラウドプラクティショナーの試験を受けるにはどうすればいいの？

認定試験は、ピアソン VUE と PSI のテスト実施業者 2 社により提供されています。

この 2 社が提供するオプションを確認の上、希望する受験場所や必要とする特定の設備などに合わせて、あなたが最も受験しやすいテストを選択しましょう。

詳細は、以下のページを参照してください。

AWS 認定試験の受験方法  
https://aws.amazon.com/jp/certification/certification-prep/testing/
